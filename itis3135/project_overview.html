Design and Proposed Project

Project Overview

  1. My application is an HTML basics website; it displays basic HTML documentation and descriptions, example/practice code, tips on how to code a presentable website (CRAP guidelines) and a practice quiz made from a form. The purpose of it is to descriptively explain/teach how to learn HTML basics and even create your own website!
  2. This website is great for users who want to learn how to create their own website/application. Along with this, it could be used by students who are studying HTML and need extra help with basic concepts.
  3. The website will consist of five pages. The first one will introduce the subject of HTML and describe what it is used for. The second page will go over basic formatting and teach users what HTML tags look like and what they do. The third page of the website will go over how to properly design a website using CRAP principles with CSS. The fourth page of the website will introduce basic Java Script concepts that will allow the user to learn how to create HTML code that is interactive. Lastly, the fifth page of the website will have a form that acts as a quiz for the user to complete, it will be based on the content of the website and display whether the user is correct or incorrect based on their answer.

Page Design

Dynamic Functionality on the Website

  • Each page of the website 1-5, will contain a photo gallery that the user is able to flip through, each photo will have a corresponding caption. On the fifth page of the website, there will be a quiz that the user submits based on what they learned throughout the website. I will implement Java Script code so that there is a notification when a question is not filled out, the submit and refresh buttons work, and once submitted, feedback to the user is given.
  • I am using photo galleries on each page in order to keep it looking professional so that the page does not seem "messy"
  • Airbnb Uses photo galleries to showcase rooms within rental homes. There are arrows on each side of the photo that allow users to flip through the pictures.
  • I think that doing a quiz at the end of the website is unique and will be interactive and skillful for the user.
  • Code Academy implements quizzes after each learning module, this allows users to test their knowledge after each exercise.
