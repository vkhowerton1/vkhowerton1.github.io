HTML Structure

HTML provides the structure for webpages using a set of elements enclosed in tags.

Common HTML Elements

These elements define the main parts of an HTML document.

  • <!DOCTYPE html>: Declares the document type and version of HTML.
  • <html>: The root element that wraps all the content of the webpage.
  • <head>: Contains metadata such as title, styles, and scripts.
  • <body>: Contains the visible content of the webpage.
  • <header>: Defines introductory content, typically for branding or navigation.
  • <footer>: Defines footer content, typically containing copyright or contact information.

Common HTML Attributes

Attributes provide additional information about elements and modify their behavior.

  • Headings: Use <h1> to <h6> to define different heading levels. As the number grows, the importance becomes less prominence.
  • Paragraphs: Use <p> to define a block of text.
  • Links: Use <a href="URL"> to create hyperlinks.
  • Images: Use <img src="image.jpg" alt="Description"> to display images.
  • Lists: Use <ul> for unordered lists and <ol> for ordered lists.
  • Tables: Use <table>, <tr>, <td> to create structured data tables.
  • Forms: Use <form> with <input>, <select>, and <button> for user input.

Common HTML Formatting

HTML provides various tags to format content.

  • Bold: Use <b> or <strong> to emphasize text.
  • Italic: Use <i> or <em> to italicize text.
  • Underline: Use <u> to underline text.
  • <section>: Defines a section in a document.
  • <article>: Defines a self-contained piece of content.

Links & Navigation

Navigation elements help users move between pages.

  • Internal vs. External Links: Internal links navigate within the same site, while external links go to other websites.
  • Opening Links: The target="_blank" attribute opens links in a new tab.
  • Anchor Links: Use <a href="#section"> to jump to sections within a page.
  • <nav>: Defines a block of navigation links.

Forms & Input Types

Forms collect user input through various fields.

  • Text Fields: Use <input type="text"> for text input.
  • Checkboxes: Use <input type="checkbox"> for selecting multiple options.
  • Radio Buttons: Use <input type="radio"> for single-option selection.
  • Dropdowns: Use <select> with <option> elements for a dropdown menu.
  • Submit & Reset Buttons: Use <button type="submit"> to submit a form and <button type="reset"> to clear fields.

HTML Practices

Following best practices improves code readability and maintainability.

  • Commenting: Use <!-- This is a comment --> to add notes without affecting the page.
  • Proper Indentation & Readability: Keep consistent indentation and use meaningful element names.

Where to Start:

You may feel overwhelmed after reading each section above, questioning yourself about where to start. Here are a few tips on how to begin:

  1. Begin by outlining a structure and creating content before coding; this will keep you organized and make your job easier
  2. Make sure to set up a code editor! Editors like VS Code, Sublime Text, and Notepad++ are great Integrated Development Environments (IDE's)
  3. Create a basic HTML file! This includes any elements mentioned in the "Common HTML Elements" segment above.
  4. Experiment! Try out different tags to see what they do, this will help you to understand what you want in your code and how you would like to display it.
